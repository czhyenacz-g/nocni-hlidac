# Obrazové assety

Sem patří zdrojová pixel-art grafika (mockupy, spritesheety, reference).

Soubory určené k servírování v prohlížeči patří do `/public/assets/images/`,
tato složka je pro zdrojové/pracovní materiály.

První prototyp používá čistě CSS placeholder grafiku (`styles/pixel.css`),
žádné obrázkové assety zatím nejsou potřeba.
